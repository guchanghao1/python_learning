1 - 关于 python 图标
    python是没有图标的，他是在pycharm里面使用的，pycharm没有配置python就不能运行

2 - 关于打开文件：
    a - 直接 左上角file -- open -- 去选择文件的位置（D盘、E盘、桌面）
    b - 直接打开文件资源管理器 -- 先找到python代码文件存放的位置 -- 然后将他的路径复制 -- 到pycharm里面 -- 左上角file -- open -- 直接粘贴

3 - 关于pycharm页面：
    页面分为 四个区域 菜单栏区、文件区、代码区、结果区
    那有些同学进来pycharm发现什么都没有，没有代码区没有文件区，那是因为都关闭了，你点击旁边的项目，或者project就好了

4 - 创建文件：
    a - 资源管理器里面创建好，到pycharm点开
    b - 文件 -- 新建项目 -- 选择文件存放地址 -- 给一个名字 -- 把解释器配置好，选择现有的就好了

5 - 关于上课时间和回放的问题：
    a - 上课时间：固定每周1 3 5 ，晚上的七点半
    b - 回放：和我们上课直播是同样的链接，点进去看我们的课程就好了
    c - 谷歌搜索引擎：有些同学说谷歌也看不了的，在设置里面把谷歌搜索引擎换了，换成其他的
                还有些同学谷歌点进来页面不是这样的，你看看你是不是下载了360安全卫士之类的，有可能被360篡改了哈

6 - 关于中文插件和pip的问题：
    a - 中文插件：演示怎么安装，怎么取消中文插件
    b - pip文件：演示怎么插入文件中

7 - 基础的快捷键：
    复制 -- Ctrl + C
    粘贴 -- Ctrl + V
    剪切 -- Ctrl + X
    全选 -- Ctrl + A
    查找 -- Ctrl + F
    替换 -- Ctrl + R
    缩进 -- Tab
    打开运行 -- Win + R
    取消缩进 -- Shift + Tab
    代码注释  -- Ctrl + /
    复制当前行 -- Ctrl + D
    返回上一步 -- Ctrl + Z
    代码格式化 -- Ctrl + Alt + L
